Conditionals allow you to display content to users based on their response to something, be it a problem or a poll, and on whether they answered correctly, incorrectly, or whether they answered at all.  

One way in which you might use conditionals in your course is to test out different versions of content on different participants.  If you want one group to see one kind of content, and another group to see different content, it can be done by setting up this environment.

To create a conditional, follow these steps:

  1. First create the question in one unit.  
  2. Then create a second unit.
  3. Create the optional results (depending on what response is given to the question) in separate components in this second unit.  
  4. Export your course by going to Tools > Export.
  5. View your course in the live version, and find the unit that contains the question.
  6. Click on "Staff Debug Info" for that question component, and copy the location ID that you see in the window that pops up.  

  7. Click to view the second unit, then click on "View unit in Studio".
  8. From Studio, note the Unit Identifer ID that displays in the Unit Location module on this page.

  9. Open your course file that you exported in a code-editing program.
  10. Find the vertical file with the Unit Identifer ID you noted from the second unit.

  11. Wrap the content that you want to have display for one response in conditional tags, like this:
    <conditional sources="i4x://StanfordOnline/OpenEdX/problem/4864f660de8a4123b148940fb95dfbc7" message="" correct="True">
  12. Be sure to close the conditional tag immediately after the content that you want to have display for this response. 
  13. The location ID that you copied from earlier is what you will use as the conditional source ID.  The message attribute can be left blank if you don't want any additional message to display when someone gives a correct or incorrect answer.  The values for the correct attribute can be either "True" or "False", and will determine which answer response will give you which resulting conditional content.
  14. Do the same thing for the other content that you want to have display for the other response(s).
  15. The final code in the vertical file will probably look something like this:
    <vertical display_name="Here is the Result">
<conditional sources="i4x://StanfordOnline/OpenEdX/problem/4864f660de8a4123b148940fb95dfbc7" correct="True">
  <html url_name="73889e3f68924289832fce23005906ba"/>
  <video url_name="0e5542ae425e439197224345e323b52d"/>
  <video url_name="6c6fe4564b4e4dfcb2be2154d0f9e267"/>
</conditional>
<conditional sources="i4x://StanfordOnline/OpenEdX/problem/4864f660de8a4123b148940fb95dfbc7" correct="False">
  <html url_name="f33404fe93ef4631afeb7530fa521536"/>
  <discussion url_name="dee3604646d146e8ae966a07562c5a60"/>
</conditional>
</vertical>
  16. Save your changes to the vertical file, zip up your course and import this back into your course by going to Tools > Import.  You should be all set after that.
