Before you can create an A/B test, you'll need to set up the group configuration you plan on using. You do this by changing the "Experiment Group Configurations" Advanced Setting. See below for sample code used in this demo course. This creates a configuration with two groups, with students split evenly between them. If you plan on having more than one group configuration, make sure the description and name are different. Otherwise, you can just copy this code into your course.

[
    {
        "description": "Content experiment 1",
        "version": 1,
        "id": 0,
        "groups": [
            {
                "version": 1,
                "id": 1,
                "name": "group 1"
            },
            {
                "version": 1,
                "id": 2,
                "name": "group 2"
            }
        ],
        "name": "Experiment 1"
    }
]
