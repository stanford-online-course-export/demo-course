categories

Discussions are by default all added to the same single category in the Discussion page: "General".  If you'd like to organize discussions by category, you can create additional categories.  To do that, click on "Settings" in the menu bar at the top of your page in Studio and select "Advanced Settings". Find the policy key for discussion_topics. 


The field on the right is editable and is where you can add the different categories you'd like to have.

By default, you are started with the following value for the "General" category:

{ "General": { "id": "i4x-StanfordOnline-OpenEdX-course-Demo" } }

Copy everything that is inside of the two outermost curly brackets and paste them after "...-Demo" }" but before the very last curly bracket.  Do this as many times as you'd like.  Change "General" to a topic title of your choosing, and append a string to the ID value, so that your new discussion topic also has a new URL that it will live under, but still within the Discussion page.  The screenshot above is an example of how this might look after adding two new discussion topics, one for "Suggestions" and another for "Advanced Topics".  Here is the code for reference:

{
    &quot;General&quot;: {
        &quot;id&quot;: &quot;i4x-StanfordOnline-OpenEdX-course-Demo&quot;
    }
    &quot;Suggestions&quot;: {
        &quot;id&quot;: &quot;i4x-StanfordOnline-OpenEdX-course-Demo_suggestions&quot;
    }
    &quot;Advanced Topics&quot;: {
        &quot;id&quot;: &quot;i4x-StanfordOnline-OpenEdX-course-Demo_advanced&quot;
    }
}

If you save your changes, you'll then be able to see these new discussion categories by heading over to the Discussion page and clicking on the arrow to the right side of where it says, "Show All Discussions".
