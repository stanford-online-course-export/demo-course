Single inline dropdown problem example instructions

If you would rather display your dropdown question in more of a sentence structure than a vertical list isolated from your question, you can use the following XML in the default visual editor of a basic common problem component.  

To do that, first click on "Problem" from the component options.  Select "Blank Common Problem" from the Common Problems list.  Click to edit this problem, and paste in the following XML, replacing everything in the visual editor:

<problem> 
<optionresponse> 
<p style="display:inline">Enter your question prompt here </p> 
<optioninput label="" options="('enter-dropdown-response-option-1-here','enter-dropdown-response-option-2-here','add-more-between-single-quotes')" correct="enter-the-correct-dropdown-response-option-here" inline="1"></optioninput> 
<p style="display:inline">.</p> 
</optionresponse> 
</problem>
