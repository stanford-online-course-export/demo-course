Create your own free text-based response questions

To create these kinds of problems in your course, you'll want to go to a unit in Studio and select the "Problem" component.  Choose the "Blank Common Problem" component from the list, then click to edit it.  Copy one of the two chunks of code lower down on this page and paste it in your blank problem component.

The first chunk is the code that was used to create the graded, free text response that has no maximum or minimum word or character requirement.  The parts you'll need to change are:

def short_response(expect, ans)
<span id="short_response_question_1"/>
var elem = $("#short_response_question_1")
var label = $("#short_response_question_1")
var description = $("#short_response_question_1")
<customresponse cfn="short_response">

and

  <p>Please enter your response here.</p>

Where you see "short_response" as the value for the def and customresponse cfn tags, replace them with an easy-to-understand name.  Make sure there are no spaces, and that the names match each other.  

The span id, var elem, var label and var description should match each other.  In the code template, they currently have a value of "short_response_question_1".  Change these values in your own questions to something else, but make sure the four values all match each other and contain no spaces.

If you have more than one of this kind of problem on a page, the names for above-noted attributes should be unique from one problem to the next.

Where you see   <p>Please enter your response here.</p>, replace the text between the <p> tags with your question prompt.

An additional location that you might want to customize is this:

<textarea style="height:150px" rows="20" cols="70"/>

This attribute specifies the size of the input field.  Change the height, rows and cols to different values to shrink or expand the space of the input field.
