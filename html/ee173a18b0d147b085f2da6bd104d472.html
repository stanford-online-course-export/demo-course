The LTI tool can be added to your course by doing two main things.  The first is by adding "lti" as an advanced module, which will allow you to see this as a component which can then be added to a unit.  If you are solely using this tool as a way of linking to external content, that should be enough.  

If, however, you want to use this Learning Tool Interoperability component to integrate with external services (such as a manual grading tool), you'll additionally need to obtain a key and secret from the LTI provider.  Once you have a key and secret provided to you from the organization, you can set this up in your course.

From Studio, click on Settings > Advanced Settings.  Note the first Policy Key in there, which is for "advanced_modules". 

Most courses will see an empty Policy Value for this key:

[]

Add "lti" between the two brackets and save your changes. Additional values added here should be separated by a comma following the closing quotation mark.

If you are integrating the LTI with an external service, you'll register this LTI by providing a key and secret in this page.  Find the Policy Key for "lti_passports".  Between the two [] brackets, paste in something that looks like the following:

"test_lti_id:b289378-f88d-2929-ctools.umich.edu:secret"

Note that test_lti_id will correspond to the LTI ID name that you give this particular tool.  The string that follows the colon will be the key.  The final string following the last colon will be the secret you were provided with.  If you are integrating with multiple external services, you'll be able to paste in multiple strings like this, as long as each one is separated by a comma.

The "advanced_modules" Policy Key refers to a component that will now display as an option when you go to add content to a unit you have created.

The required fields to edit are:

Display Name - what you want this component's name to display as
launch_url - where you want this component to link people to
lti_id - a string of characters (letters or numbers, no spaces) to uniquely identify this particular LTI component

In some instances - when you want to utilize the greater functionality of LTI, the lti_id value will be a string provided to you by the external service you are integrating your course with.
