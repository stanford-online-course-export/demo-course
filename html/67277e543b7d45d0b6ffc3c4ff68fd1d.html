Inside this section, we have four subsections but only three will show in the lefthand column.  You are currently viewing the second subsection, and cannot see the first because the first subsection has been hidden! To see the hidden subsection with three units, click here.  In the image below, you can see the subsection that is hidden, but you don't see it noted in the section to the left.

How is this done?  First, you create a subsection and add a unit (or units) to it, and then add a component (or multiple components) to that unit.  Publish the unit.

The following steps were updated on 10-22-14 due to changes in the platform. 

While still viewing the unit, hover your mouse over the subsection link in the breadcrumb above your unit.  Note the corresponding URL in the browser status bar.  



The end string of that URL is the location ID for this subsection. Note the first three characters of that URL, which will follow the word "sequential".  

After you've made a note of this string, click on "Tools" in the menu bar and select "Export".  This will take you to a page in Studio where you can then export your entire course as a special zip file.  Once that has downloaded to your computer, click to unzip this file and open it in a text-editing tool.  If you have a Mac, one tool you might use is TextWrangler, which handles code edits in a very neat and orderly way.  You could also edit this in something like SublimeText, TextMate, TextEdit or NotePad.

Go to your exported course file, and go to the sequential folder.  The first three characters of the URL you noted earlier will tell you which sequential file to edit in your exported course file.

Open the sequential file in your editing program, and you'll see the XML file start with something like this:

<sequential display_name="Subsection 1">

All units you have added to this subsection will display beneath this tag as <vertical> tags. 

Add the following information to the <sequential> tag in the file:

hide_from_toc="true" start="2013-01-28T22:33:00Z"

The end result is this:

<sequential display_name="Subsection 1" hide_from_toc="true" start="2013-01-28T22:33:00Z">
   <vertical url_name="c74ed93121e6470c8b4a9d954bfc4a66"/>
   <vertical url_name="305fd6726c94406a860cc6d2ff6b0c38"/>
   <vertical url_name="d669b491f89048d3996ccc199e26275e"/>
</sequential>

 Here's a screenshot of how my sequential file looks now:

 After you've edited your file, save your changes.  Then, zip this file up and import it back into your course.  

The subsection will no longer show up in the Courseware column or anywhere in the course, but if you add a link to it in your course, then people will be able to access the content via the link you provide.

Clicking the link will take you to the hidden subsection. 

In order to retrieve the link to the hidden subsection, go back to Studio and click on a unit in the subsection.  Click on the "View Live Version" button.  Copy the URL.  You can use this URL to access your hidden subsection at any time.
