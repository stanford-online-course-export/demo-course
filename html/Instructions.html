Adding a Table of Contents to a Video

To add a Table of Contents to your video, first you'll have to open your video's timed .srt caption file in a text editor like TextWrangler. Change the timestamps and captions in the file to what you want them to be for your Table of Contents.

For example, in the sample video in the previous unit, the original transcript had this format: 

0
00:00:00,640 --> 00:00:02,440
In this section, I want to get to the point where you



1


00:00:02,440 --> 00:00:05,850


can write some code, and run it, and see what it does.






2


00:00:05,850 --> 00:00:08,010


Everything in the computer really comes down


.


.


.




To create the Table of Contents .srt file, the timestamps and captions were changed to look like this:


0


00:00:00,640 --> 00:02:00,580


Introduction






1


00:02:00,580 --> 00:04:11,150


Numbers






2


00:04:11,150 --> 00:04:58,660


Strings


.


.


.


(You can download the ToC .srt file to use as a reference here.)


Once you've saved your changes as a new .srt file, you have to upload the file to your video. Click "Edit" on the video component, then go to the Advanced tab. Scroll to "Transcript Languages" and click "Add". Select "Table of Contents" and upload the new .srt file. If you want the Table of Contents to display by default, make sure the "Show Transcript" setting is set to True.
