To create the above kind of problem, first create your problem with a checkbox problem component.  When you have it built the way you want it, then click on the "Advanced Editor" link.  From this window, paste in this script just after your question prompt:

  <script type="text/python" system_path="python_lib">
def hint_fn1(answer_ids, student_answers, new_cmap, old_cmap):
    aid = answer_ids[0]
    ans = ''
    hint = ''
    if not aid in student_answers:
        hint = 'Looks like you didn\'t select an answer?'
    else:
        ans = str(student_answers[aid]).lower()
    print 'hint_fn called, ans=', ans
    if 'choice_0' in ans:
        hint = 'This hint will display to anyone who checks the box next to the first response option.'
    if 'choice_1' in ans and 'choice_1' not in ans:
        hint = 'This hint will display to those who check the second response option but not the first response option.'
    if 'choice_2' in ans:
        hint = 'You don\'t need to have a hint for each response option.'
    if 'choice_3' in ans:
        hint = 'This hint would display if there is a fourth response option and the box next to it is checked.'
    if hint:
        hint = "&lt;font color='blue'&gt;Hint: {0}&lt;/font&gt;".format(hint)
        new_cmap.set_hint_and_mode(aid,hint,'always')
  </script>

Your response options are noted in the script above as choices.  'choice_0' corresponds to the first response option in your question. 'choice_1' equates to the second response option, and so on. Note that most characters that are not letters will not be accepted.  If you have a word like "don't" in your hint, make sure to write it like so:

don\'t

Before your closing </choiceresponse> tag, paste in this single line of code:

    <hintgroup hintfn="hint_fn1"/>
