Multiple inline dropdown problem example instructions

If you want to have more than one dropdown problem in a single component, you can use the above code, but add another <optioninput> tag inside of the <optionresponse></optionresponse> tags.  The added pieces are highlighted in red below.  This might look like this:

<problem> 
<optionresponse> 
<p style="display:inline">Enter your first question prompt here </p> 
<optioninput label="" options="('enter-dropdown-response-option-1-here','enter-dropdown-response-option-2-here','add-more-between-single-quotes')" correct="enter-the-correct-dropdown-response-option-here" inline="1"></optioninput> 
<p style="display:inline">and if you want a second dropdown problem in the same question, add the second part of your question prompt here</p> 
<optioninput label="" options="('and-enter-dropdown-response-option-1-here','and-enter-dropdown-response-option-2-here')" correct="and-enter-the-correct-dropdown-response-option-1-or-2-here" inline="1"></optioninput> 
<p style="display:inline">and end your question here, as you see fit.</p> 
</optionresponse> 
</problem>
