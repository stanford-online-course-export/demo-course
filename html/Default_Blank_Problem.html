Default Blank common PRoblem

Blank Common Problem components are just that: blank.  A blank common problem contains the <problem> tags that wrap around your problem, and "Submit" and "Show Answer" buttons, but other than, it's a clean blank template to build your problem in. 

You can create your own problem with markdown language or with the provided icons in the problem component toolbar.

See this page for a detailed explanation of the problem component toolbar.
